### Bloque de código  
Para mostrar un bloque de código que se pueda copiar para usar en otro lado, como el siguiente:  

```bash
sudo reboot
```
**La sintaxis es:**  
Usando tres acentos graves ( \` \` \` ) al principio y al final.  

**Lenguaje:** Al escribir `bash` justo después de los primeros tres acentos, le indicas a GitHub que aplique colores automáticos a los comandos (resaltado de sintaxis).  

**Código en línea:** Para que la palabra `sudo` en el segundo párrafo aparezca dentro de un pequeño recuadro gris, se envuelve entre acentos graves simples ( \`sudo\` ).  

**Botón de copiar:** No necesitas programar el botón. GitHub lo agrega **automáticamente** en la esquina superior derecha de cualquier bloque de código cuando detecta la sintaxis de los tres acentos graves.
